# webcomp-2018
### 大学の講義 (http://www.cl.ecei.tohoku.ac.jp/index.php?Web-Computing) で作成したウェブアプリです.
### 実際に使用するときは，`app.py`と同じディレクトリ上に以下のような内容の`.env`ファイルを作成してから実行してください．

> API_KEY = "(Google Cloud PlatformのAPI Key)"
